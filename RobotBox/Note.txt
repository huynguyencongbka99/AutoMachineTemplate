Nếu theo mô hình ABB ↔ C# thì luồng xử lý thường như sau:

PC/HMI
   ↓
Send Command
   ↓
Robot nhận lệnh
   ↓
ACK
   ↓
Robot thực thi
   ↓
DONE hoặc ERROR

Ví dụ lệnh Home:

PC  ---------- CMD:HOME ---------> Robot
PC  <--------- RSP:ACK ---------- Robot

Robot chạy về Home

PC  <--------- EVT:DONE --------- Robot


Phía C#

Ví dụ người dùng bấm nút Home:

private async void btnHome_Click(object sender, EventArgs e)
{
    bool ack = await robot.SendCommandAsync("CMD:HOME\n");

    if (!ack)
    {
        MessageBox.Show("Robot không phản hồi");
        return;
    }

    Console.WriteLine("Robot đã nhận lệnh");
}

ReceiveLoop chạy nền
private void ProcessMessage(string msg)
{
    switch(msg)
    {
        case "RSP:ACK":
            _ackWaiter?.TrySetResult(true);
            break;

        case "EVT:BUSY":
            State = RobotState.Busy;
            break;

        case "EVT:DONE":
            State = RobotState.Idle;
            break;

        case "EVT:ERROR":
            State = RobotState.Error;
            break;
    }
}

Nếu muốn đợi robot chạy xong

Tạo thêm:

private TaskCompletionSource<bool> _doneWaiter;

Nhận DONE:

case "EVT:DONE":
    _doneWaiter?.TrySetResult(true);
    break;

Hàm:

public async Task<bool> WaitDoneAsync(int timeout = 30000)
{
    _doneWaiter = new TaskCompletionSource<bool>();

    Task completed = await Task.WhenAny(
        _doneWaiter.Task,
        Task.Delay(timeout));

    return completed == _doneWaiter.Task;
}

Sử dụng:

bool ack = await robot.SendCommandAsync("CMD:HOME\n");

if (!ack)

bool done = await robot.WaitDoneAsync();
